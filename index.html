Nginx

简介

nginx [engine x] 是一个集HTTP、反向代理、邮件代理 和 UTCP/UUDP 的服务器。第一个版本由 Igor Sysoev 编写。很长一段时间后, Nginx 开始在俄国的一些高负载网站上运行包括：Yandex, Mail.Ru, VK, 和 Rambler等。根据 Netcraft 公布的2017年八月的调研数据, 大约 29.25% 的商业网站使用 Nginx 服务器。著名的案例有: Netflix, Wordpress.com, FastMail.FM.

Nginx 的源代码和文档遵循 2-clause BSD-like 协议。

由 Nginx, Inc 提供商业支持。

基本 HTML 服务器特性：
1.自动索引静态文件和 index 文件；文件打开句柄缓存；
2.反向代理缓存加速; 负载均衡和故障容错；
3.缓存加速支持 FastCGI, uwsgi, SCGI, 和 memcached 服务器; 负载均衡和故障容错；
.模块架构. 过滤器包括： gzipping, byte ranges, chunked responses, XSLT, SSI, 和 image transformation 等过滤器. Multiple SSI inclusions within a single page can be processed in parallel if they are handled by proxied or FastCGI/uwsgi/SCGI servers; .SSL and TLS SNI ;
6.Support for HTTP/2 with weighted and dependency-based prioritization.
